About Sixteen

Sixteen is a Crystal library and CLI tool for working with Base-16 color themes following the Tinted Themes specification.

Features:

  • Access 300+ Base-16 theme YAML files embedded in the binary
  • Generate theme CSS with custom templates
  • Auto-generate theme variants (dark â†” light)
  • Browse themes interactively in the terminal
  • Export themes in multiple formats

GitHub: https://github.com/ralsina/sixteen
